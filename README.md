# Oconsul Website

This repo provides the code for the Oconsul website based on Bootstrap and built for Firebase. To work with the website, you must install the Firebase cli. You must have access to our Firebase account to push updates via the cli.

Oconsul is based on the Bootstrap Stylish Portfolio created by Start Bootstrap.

[Stylish Portfolio](http://startbootstrap.com/template-overviews/stylish-portfolio/) is a responsive, one page portfolio theme for [Bootstrap](http://getbootstrap.com/) created by [Start Bootstrap](http://startbootstrap.com/). The theme features multiple content sections with an off canvas navigation menu.
